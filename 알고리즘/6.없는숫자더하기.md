두번째 풀이

```
function solution(numbers) {
    var answer = 0;
    for (let i=1; i<=9;i++) {     //0은 더해도 변하지 않으니 1부터 9까지 반복문 
        if (!numbers.includes(i)) { //numbers에 i가 포함되지 않는다면
            answer +=i;             // 그 i를 answer에 계속 더하기!
        }
    }
    return answer;
}
```

.includes() : 포함하는지 안하는지 알 수
